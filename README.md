## Accessing the server online
The backend server used in this training can be accessed at https://awesome-store-server.herokuapp.com/. You can access the data via various endpoints. For example, to retrieve the reviews for the product with id = 1, we can hit the endpoint...
```
http://localhost:4201/products/1/reviews
```
